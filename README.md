# customer-churn-dashboard
Interactive Customer Churn Dashboard for analyzing churn trends, customer behavior, and key retention metrics using data visualization and business intelligence techniques.

## Objectives
- Analyze customer churn trends.
- Identify key factors influencing customer churn.
- Monitor customer demographics and account information.
- Provide interactive visualizations for business decision-making.

## Dashboard Features
- Overall customer and churn summary
- Churn rate analysis
- Customer demographics 
- Contract type analysis
- Payment method analysis
- Internet service analysis
- Monthly and total charges analysis
- Interactive filters and slicers
- KPI cards for quick insights

## Technologies Used
- Power BI (or Tableau/Excel, depending on your project)
- Microsoft Excel / CSV Dataset
- Data Cleaning and Transformation
- Data Visualization

## Dataset
The dataset contains customer information such as:
- Customer ID
- Gender
- Tenure
- Internet Service
- Contract Type
- Payment Method
- Monthly Charges
- Total Charges
- Churn Status

## Key Insights
- Month-to-month contract customers have the highest churn rate.
- Customers with higher monthly charges are more likely to churn.
- Electronic check payment users show higher churn.
- Longer-tenure customers are less likely to leave.
- Fiber optic internet users have comparatively higher churn.

## How to Use
1. Download the Power BI (.pbix) file.
2. Open it using Microsoft Power BI Desktop.
3. Explore the dashboard using filters and slicers.
4. Analyze customer churn patterns and business KPIs.
